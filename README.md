# Macfile Remover
A simple app for removing hidden files created by macOS.
  
## Installation
```bash
git clone https://github.com/MisterChamster/private-appledouble-remover.git macfile-remover
cd macfile-remover
```

## Usage
```bash
#Windows
python main.py
```
```bash
#Linux/macOS
python3 main.py
```
